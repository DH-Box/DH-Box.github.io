Get Started


DH Box is currently under development. Follow our blog to find out the latest, including when DH Box will be available for use.


DH Box is made available to users as a hosted service. Please make sure to create a back-up system for all important files you will create and intend to use beyond the duration of your DH Box.


If you're interested in hosting your own virtual computer, please check out our documentation on GitHub.
